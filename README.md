## в—Ҳ What Is This?

This is not a typical portfolio. It's a **living, breathing knowledge graph** вҖ” each node is a project, positioned in 3D space by a custom force-directed physics simulation. Nodes repel each other, edges pull same-category projects together, and the whole system settles into a layout that reveals the structure of the work itself.

Every interaction is intentional:
- **Hover** reveals project metadata and direct links
- **Click** opens a panel with GitHub + live demo buttons
- **Search** fades out non-matching nodes in real time
- **Category filter** isolates domains by clicking the legend

## в—Ҳ Physics Simulation

The layout is computed by a **custom 3D force-directed algorithm** вҖ” no D3, no physics library.

```
Each frame (pre-baked, 150 iterations):

  REPULSION   Fбөўвұј = 900 / |rбөў - rвұј|ВІ    (nodes push apart)
  ATTRACTION  Fбөўвұј = (d - 28) Г— 0.018     (linked nodes pull together)
  GRAVITY     F   = -0.012 Г— position     (centering force)
  DAMPING     v   = v Г— 0.78              (energy dissipation)
```

Result: same-category nodes cluster naturally. Cross-category edges create bridges. Complexity score controls node radius, making heavier projects visually dominant.

## в—Ҳ Customization

**Add a project** вҖ” append to the `RAW` array in `<script>`:
```js
{
  name: "MyProject",
  score: 150000,          // complexity score (controls node size)
  gh: "https://github.com/salonyranjan/MyProject",
  demo: "https://my-project.vercel.app/"  // null if no demo
}
```

**Change category rules** вҖ” edit the `classify()` function:
```js
function classify(name, score) {
  if (name.includes('myterm')) return 'ai';
  // ...
}
```

**Adjust physics** вҖ” tune constants in `runForce()`:
```js
const f = 900 / d2;          // repulsion strength
const f = (d - 28) * 0.018;  // link rest length & stiffness
n.vx *= 0.78;                // damping (lower = bouncier)
```
